# PLACIDA

## Project Description
Placida is a human-centered mental health companion built using design thinking. It enables users to track moods, manage stress, and access support through journaling, breathing exercises, and a simple chatbot. Developed using insights from user personas, empathy maps, and journey analysis.

## Tech Stack
- HTML
- CSS
- Vanilla JS
- localStorage
